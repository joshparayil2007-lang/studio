# **App Name**: LocalListings

## Core Features:

- User Authentication & Profiles: Allow users to register, log in, and manage basic profile information necessary for posting and communication.
- Listing Creation & Management: Enable users to create new classified ads with descriptions, details, and multiple photo uploads. Users can also view and manage their active and past listings.
- Categorized Browsing: Users can intuitively browse listings across various predefined categories and subcategories, similar to existing classifieds platforms.
- Advanced Search & Filtering: Provide a powerful search functionality with keywords, location-based filtering, and additional filters like price range, condition, or listing type.
- Secure In-App Messaging: Facilitate secure, private messaging between interested users and listing owners for inquiries and negotiations directly within the app.
- AI Listing Assistant: An AI tool that suggests relevant categories, generates engaging titles, and refines ad descriptions based on the user's input and provided details.

## Style Guidelines:

- Light color scheme with a calming primary green. Primary Color: #50A63F, Background Color: #F0F4F0, Accent Color: #B7DB70. The primary green provides a sense of natural reliability, contrasted by a near-white background for visual clarity, and accented by a brighter yellow-green for interactive elements.
- Body and headline font: 'Inter' (sans-serif) for a modern, objective, and highly legible appearance suitable for extensive text across listings and messages.
- Utilize modern, clean line icons that convey functionality without visual clutter, complementing the minimalist design aesthetic.
- Employ a grid-based, responsive layout with generous whitespace to ensure readability and an uncluttered user experience on all device sizes.
- Incorporate subtle and smooth transitions for page navigation, search result loading, and interactive elements to enhance user feedback and overall fluidity.
